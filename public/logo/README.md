# Логотипы Kolibel

Сохрани сюда 3 PNG-файла **с такими именами** (с прозрачным фоном):

| Файл                  | Что это                                                    | Где используется                                               |
| --------------------- | ---------------------------------------------------------- | -------------------------------------------------------------- |
| `kolibel-full.png`    | Значок + надпись «Kolibel» справа (горизонтальный логотип) | Хедер сайта                                                    |
| `kolibel-stacked.png` | Значок сверху, «Kolibel» снизу (вертикальный логотип)      | Футер во всех SupportLinks                                     |
| `kolibel-mark.png`    | Только значок (без текста)                                 | Бейдж рядом с именем сотрудников Kolibel и у названия компании |

Vite раздаёт `public/` из корня, поэтому в коде они доступны как
`/logo/kolibel-full.png`, `/logo/kolibel-stacked.png`, `/logo/kolibel-mark.png`.
